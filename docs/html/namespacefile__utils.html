std::vector< fs::path > file_utils::dir ( const std::string &  pattern)

Returns a list of paths matching the specified pattern that can contain *, ? (Windows, POSIX) and [..] (POSIX).

Parameters
patternthe pattern to use for search files
Returns
the list of matching files

Ending path separators are ignored. Wildcards are only allowed in the filename part of the path.

Warning
Wildcard [..] support is not yet implemented.
