file-utils-static static library

This library provides few file functions to check EOL (end of line) mode, list files matching a pattern and concatenate files.

To implement it in your project, install the overlay port file-utils-static and use vcpkg. Set your path to overlay ports in vcpkg-configuration.json ("overlay-ports": "your_path_to_overlay_ports").

Full documentation of function and class at Doxypress documentation.
